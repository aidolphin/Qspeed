# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 1.x (current) | ✅ |

## Reporting a vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Email: quantumleaf@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 48 hours. We take all reports seriously and will credit researchers in the changelog.
